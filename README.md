# TrainTickets Booking System

A simple traintickets booking system built with Spring Boot.

## Tech Stack

* Java
* Spring Boot
* Spring Security
* Spring Data JPA / Hibernate
* Thymeleaf
* MySQL
* Maven

## Features

* User registration and login
* Search train trips by departure, destination and date
* Book train tickets and manage orders
* Seat selection by coach
* Trip reviews
* Admin management for trains, trips, tickets and users

## Run

```bash
mvn spring-boot:run
```
url: http://localhost:8081/trainticket/user/home (Home)
